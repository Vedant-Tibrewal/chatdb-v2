# Sales / E-commerce Dataset

Placeholder for the sales/e-commerce base dataset.

This directory will contain CSV files with sample sales data (orders, products, customers, etc.) that get cloned into session-scoped schemas at session creation time.

## Expected Tables
- `orders` — order_id, customer_id, product_id, quantity, total, order_date
- `products` — product_id, name, category, price, cost
- `customers` — customer_id, name, email, region, signup_date
